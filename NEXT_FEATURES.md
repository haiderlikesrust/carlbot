## üí° **Quick Implementation Tips**

- **Real-time:** Use Socket.io (easy integration with Express)
- **Search:** Use SQLite FTS (Full-Text Search) or add Elasticsearch later
- **Polls:** Simple database schema (poll_options table)
- **Reactions:** Extend existing likes table
- **Blocking:** Add `blocked_users` junction table
- **Editing:** Add `edited_at` timestamp, `edit_history` JSON column
